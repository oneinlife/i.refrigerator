# Настройка формулы product_name в Google Sheets

## Важная информация

**🎉 Автоматическое добавление формулы:** Когда вы добавляете продукты в список покупок через приложение, формула `product_name` добавляется автоматически! Вам нужно настроить формулу вручную только для существующих строк.

## Для таблицы ShoppingList

### Опция 1: Автоматическое добавление (Рекомендуется)

Просто используйте приложение для добавления продуктов в список покупок - формула будет добавлена автоматически в столбец K.

### Опция 2: Ручное добавление для существующих строк

Если у вас уже есть записи в ShoppingList без формулы в столбце K, выполните следующие шаги:

### Шаг 1: Добавьте столбец с заголовком

1. Откройте вашу Google Sheets таблицу
2. Перейдите на лист **ShoppingList**
3. В ячейку **K1** добавьте заголовок: `product_name`

### Шаг 2: Добавьте формулу

В ячейку **K2** вставьте следующую формулу:

```excel
=IFERROR(INDEX(Products!B:B; MATCH(B2; Products!A:A; 0)); "Unknown Product")
```

**Примечание:** Используется точка с запятой (`;`) как разделитель аргументов для русской локали Google Sheets.

### Шаг 3: Скопируйте формулу вниз

1. Выделите ячейку **K2**
2. Наведите курсор на правый нижний угол ячейки (появится маленький синий квадратик)
3. Потяните вниз, чтобы скопировать формулу на все строки с данными
4. Или нажмите **Ctrl+C** (скопировать), выделите диапазон K3:K1000 и нажмите **Ctrl+V** (вставить)

## Как работает формула

- **MATCH(B2; Products!A:A; 0)** — находит позицию product_id (из столбца B) в столбце A таблицы Products
- **INDEX(Products!B:B; ...)** — возвращает название продукта (из столбца B таблицы Products) по найденной позиции
- **IFERROR(...; "Unknown Product")** — если продукт не найден, выводит "Unknown Product"
- **Точка с запятой (;)** используется как разделитель аргументов в русской локали Google Sheets

## Альтернативные формулы

### Вариант 2: VLOOKUP
```excel
=IFERROR(VLOOKUP(B2; Products!A:B; 2; FALSE); "Unknown Product")
```

### Вариант 3: XLOOKUP (если доступен)
```excel
=XLOOKUP(B2; Products!A:A; Products!B:B; "Unknown Product")
```

## Преимущества

✅ Автоматическое отображение названия продукта в списке покупок  
✅ Быстрая загрузка данных (меньше операций JOIN в коде)  
✅ Не нужно делать дополнительные запросы к таблице Products  
✅ Офлайн режим работает лучше (название продукта сохраняется в кэше)

## Проверка

После добавления формулы:
1. В столбце K должны появиться названия продуктов
2. Для несуществующих product_id будет показано "Unknown Product"
3. При добавлении новой записи формула автоматически заполнится (если используете копирование)

## Обслуживание

- Формула автоматически обновляется при изменении названий в таблице Products
- При добавлении новых записей вручную, не забудьте скопировать формулу в новые строки
- Можно настроить автоматическое копирование формулы через Google Apps Script (опционально)
